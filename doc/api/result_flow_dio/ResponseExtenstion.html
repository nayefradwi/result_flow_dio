ResponseExtenstion

ResponseExtenstion<T> extension

ResponseExtenstion is an extension on the Response class from Dio. It provides a method to extract a Result object from the response. The withResult method returns a new Response object with the data transformed to a Result type.

This is useful for handling the response in a more functional way, using the Result type to represent success and failure.

The result getter extracts the result from the response. If the data is already a Result, it returns it as is. Otherwise, it wraps the data in a Result.success object.
