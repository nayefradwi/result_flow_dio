ResponseAsyncExtension

ResponseAsyncExtension<T> extension

ResponseAsyncExtension is an extension on the Future<Response<T>> class from Dio. It allows to safely extract FutureResult from an async response.

The unSafeResult getter extracts the result from the response. If the Future<Response> throws an exception, it will NOT be caught.

The result getter safely extracts the result from the response. If an excpetion occurs, it returns a Result with the error as UnknownError.
